# Flask To-Do (ПР3)

Простое приложение "Список дел" на Flask.

## Функции
- Просмотр списка задач
- Добавление новой задачи
- Удаление задачи
- Очистка всех задач
- Автоматическая дата создания для каждой задачи
- Хранение задач в `tasks.json`

## Запуск
1. Установить зависимости:
   `pip install -r requirements.txt`
2. Запустить приложение:
   `python app.py`
3. Открыть в браузере:
   `http://127.0.0.1:5000`
